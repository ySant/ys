My Projects

There are four projects, so far, that I have posted on GitHub. You can get the source code through the link on the left.

Clicking Game - Python - TkInter

In this project I made a game with Python and the Python TkInter library. In the game, there are one hundred buttons placed in an array. When the game starts, the buttons are labelled by numbers from 0 to 99. The player is asked to click the buttons sequentially. There is a time limit to click all the buttons.

Cloth Animation - C++ and OpenGL

In this project, I implemented cloth animation using spring mesh. I used FreeGlut library to create the window and drive the animation.

Snake Cube Puzzle Solver - Java

The snake cube puzzle consists of 27 small cubes that are strung together by a flexible string. The string goes through each cube either in a straightforward or an elbow way. The problem is how to shape the snake into a 3x3x3 cube. I wrote a code to solve this puzzle by using the backtracking algorithm - implemented in Java.

Physics Animation - C++ and OpenGL

This is a collection of simple physics animations. So far, I have simple pendulum, simple spring, spring pendulum, and double pendulum.
